# vim-config

My VIM setup

# Requirements

* neovim >= 0.5.0

# Installation

    git clone https://github.com/sinisa-susnjar/vim-config.git

    cd vim-config

    git submodule update --init --recursive # for Vundle

    ln -s vim-config/vimrc ~/.vimrc
    ln -s vim-config/vim ~/.vim
    ln -s vim-config/nvim ~/.config/nvim
